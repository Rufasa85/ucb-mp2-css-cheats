CSS Cheat Sheet

Just some of my favorite CSS snippets. Feel free to use wherever you like!

CSS Variables

If I have a bunch of reused CSS, I can create some variables to reduce potential errors when redesigning 

:root {
    --primary-bg: slategray;
    --primary-text-color: deepskyblue;
    --accent-color: white;
}

Veritical alignment

Thought vertical alignment was impossible? To quote Celtics great Kevin Garnett, ANYTHING IS POSSIBLE (with flexbox)! 

.flex-box {
    display: flex;
    justify-content:center;
    align-items:center;
}

Cool 2 tone gradient

If you want your app to look like the 90s, try out this gradient:

.card {
   background: linear-gradient(
    120deg, 
    blue 0%, 
    blue 50%, 
    pink 50%, 
    pink 100%
   )
}
